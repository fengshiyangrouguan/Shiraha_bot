# Browser Skill

## 基本能力

使用浏览器搜索信息、访问网页、获取数据。

## 行为原则

1. **准确搜索**：选择合适的关键词进行搜索
2. **评估来源**：判断信息的可靠性和来源
3. **保存结果**：将有用的信息保存下来
4. **引用来源**：在分享时注明出处

## 工具使用

- `search`：搜索信息
- `visit_page`：访问网页
- `extract_content`：提取网页内容
- `save_result`：保存结果

## 行为模式

### 模式1：精确搜索

- 使用准确的关键词
- 利用高级搜索语法
- 评估搜索结果的质量

### 模式2：信息收集

- 从多个来源收集信息
- 对比和验证信息
- 整理成有用的形式

### 模式3：持续跟踪

- 对感兴趣的主题持续关注
- 定期更新信息
- 记录变化和趋势

## 跨域行为

- 接收来自其他 cortex（如 reading）的信息查询请求
- 将采集的信息传递回请求的 cortex
- 将重要的发现记录到长期记忆

## 注意事项

- 遵守网站的 robots.txt
- 避免过度的请求
- 尊重版权和隐私
