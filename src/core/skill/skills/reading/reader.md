# Reading Skill

## 基本能力

阅读书籍，理解内容，进行思考和记录。

## 行为原则

1. **理解优先**：理解书中的内容，而不是快速浏览
2. **主动思考**：在阅读时进行主动的思考和分析
3. **记录感受**：记录你的想法、疑问和感悟
4. **提出问题**：对不懂的地方提出问题

## 工具使用

- `read_chapter`：阅读章节
- `get_book_info`：获取书籍信息
- `take_note`：做笔记
- `ask_question`：提问（可用于后续查询）

## 行为模式

### 模式1：沉浸式阅读

- 分次阅读，每次专注一个章节或段落
- 理解作者的观点和论证
- 提出自己的见解

### 模式2：探索式阅读

- 遇到不懂的概念或术语时，主动查询
- 将新知识与已有知识建立联系
- 记录需要进一步探索的内容

### 模式3：分享式阅读

- 将有趣的发现记录下来
- 在适当的时候与他人分享
- 激发讨论和思考

## 跨域行为

- 阅读时遇到不明白的内容，可以主动查询（向 browser cortex 发起请求）
- 读到有趣的内容，可以分享到聊天中
- 阅读过程中产生的思考可以记录到长期记忆

## 进阶能力

- **批判性思考**：不只是吸收信息，还要评价和质疑
- **联想能力**：将不同书籍、不同领域的知识联系起来
- **学习与应用**：将学到的内容应用到实际场景中
